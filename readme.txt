Ready to use WhatsApp notifications service plugin for WooCommerce. Install and start sending.

== Description ==

Do you have awesome ecommerce buisness with hundreds of buyers everyday? Need more close connection with your users with WhatsApp? You are on at a right place.
Install plugin and start sending WhatsApp messages about your orders.
The plugin is evolving daily to match need of more and more sites. Just install and start sending notifications.

==How to use WooNotify==
	
1. Install WooNotify.
2. Activate Plugin.
3. Goto settings and customize your template.
4. Save settings
5. And you done!
	

== How to send WhatsApp notifications ==

The plugin support two methods as of now. Use according to your need.

**Manual**
When customer purchases something you will see pending notifications in admin bar. Click on icon to send those notifications using web whatsapp.

**Automatic**
This way you dont need to do anything your buisness account will be hosted with us. Notifications will be sent automatically.

== Installation ==

Use built in automatic installer or,

1. Copy the plugin folder into /wp-content/plugins
2. Activate plugin via the plugins page in the WordPress admin
3. Go to Settings to configure.
